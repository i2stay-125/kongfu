# KungFuBuy - Global Dropshipping Platform

A modern, responsive dropshipping platform website built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- 🏠 **Home Page** - Hero section with Enterprise/Individual service cards
- 🏢 **Enterprise Page** - B2B service platform features
- 🛍️ **Individual Page** - B2C shopping platform features
- 📋 **Process Page** - 4-step dropshipping workflow guide
- 📞 **Contact Page** - Contact form and company information
- 🔐 **Login Page** - Secure admin authentication (admin/123456)

## Design

- **Primary Color**: #16A34A (Green)
- **Hover Color**: #15803D (Darker Green)
- **Background**: #F9FAFB (Light Gray)
- **Font**: Inter

## Getting Started

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Pages

| Route | Description |
|-------|-------------|
